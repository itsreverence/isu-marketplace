# Common Weakness Enumeration Project

## Brief Project Description: Marketplace Application

The group project we built is a marketplace application where users can list things for sale as well as purchase things others have listed

The Marketplace allows users to perform a variety of actions, such as:

```
- Register or login to an account
- View the listings that you have created
- Create a new listing on the marketplace
- Remove one of your listings from the marketplace
- Browse all the listings other people have created
- Search for listings on the marketplace
- Buy a listing that someone else has created
- View the information of all the users (Admin)
- Update the role of a specified user (Admin)
- Delete a user and their listings (Admin)
```

## Run the project

The app includes helper scripts that compile and run the code.

### Windows

```powershell
.\run.cmd
```

### macOS/Linux

```bash
chmod +x run.sh
./run.sh
```

Both scripts compile all Java files and run the driver using the included jar files
